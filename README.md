## ⚙️ Installation 

If you would like to try this application or want to use this as a demo, please clone the repo, and run the following command on the terminal

``` 
npm install
``` 

## 🖥️ Usage 

Start the server using the following command on the terminal
``` 
npm run develop
``` 
Runs the app in the development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
